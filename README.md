Event Registration

Project Description

Event Registration is a simple blockchain-based smart contract project built using Solidity. It allows anyone to set up an event and participants to register for it directly on the blockchain. This project is perfect for beginners who want to learn how to build and deploy smart contracts while experimenting with decentralized event management.

What It Does

This smart contract allows:

Setting event details such as name, date, and maximum participants.
Registering participants for the event.
Viewing the total number of participants.
Everything is stored on-chain, making it secure, transparent, and immutable.

Features

✅ Simple and beginner-friendly
✅ Anyone can set event info
✅ Anyone can register
✅ View total registered participants
✅ Fully decentralized
